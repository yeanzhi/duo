## 特点

  1. 对 Javascript、 HTML 和 CSS 一流的支持
  2. 使用 unix-y 一样的命令行界面
  3. 直接从 GitHub拉取代码源文件或者相应版本
  4. 支持源转换，如 Coffeescript 或SASS
  5. 不需要一个清单文件
